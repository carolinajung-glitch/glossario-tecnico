HTML & CSS

HTML e CSS são as duas tecnologias fundamentais para a criação de páginas web. O HTML (HyperText Markup Language) é usado para estruturar o conteúdo da página, definindo elementos como títulos, parágrafos, imagens, etc. Já o CSS (Cascading Style Sheets) é usado para estilizar a aparência da página, controlando cores, fontes, layout e outros aspectos visuais.
